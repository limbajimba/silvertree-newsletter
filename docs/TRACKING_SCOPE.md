## 7. Intelligence Focus Areas

### 7.1 Deal Types Monitored

| Deal Type | Priority | Description |
|-----------|----------|-------------|
| **M&A Acquisition** | High | Company acquiring another |
| **M&A Merger** | High | Merger of equals |
| **Divestiture** | High | Business unit sale (carve-out source) |
| **Fundraising** | Medium | VC/PE investment rounds |
| **IPO** | Medium | Public offerings |
| **Strategic Partnership** | Medium | Major alliances |
| **Product Launch** | Low | New products/features |
| **Personnel Change** | Low | Executive appointments |

### 7.2 Carve-Out Opportunity Detection

The system specifically screens M&A deals for carve-out signals:

#### Trigger Signals

| Signal | Description |
|--------|-------------|
| **"Non-core" language** | Explicit mention of non-core divisions |
| **Rationalization** | "Streamlining" or "rationalization" messaging |
| **Focus statements** | Acquirer focusing on "core" business areas |
| **Geographic misfit** | Units in regions outside acquirer strategy |
| **Product misfit** | Product lines outside acquirer's focus |
| **Legacy assets** | Older products being deprioritized |

#### Carve-Out Assessment Criteria

| Potential | Criteria |
|-----------|----------|
| **High** | Clear non-core unit + strategic fit for SilverTree portfolio |
| **Medium** | Possible opportunity, worth monitoring for developments |
| **Low** | Unlikely but noted for completeness |
| **None** | No carve-out potential detected |

### 7.3 Competitive Threat Signals

The system identifies competitive threats through:

| Signal Type | Examples |
|-------------|----------|
| **Market Entry** | Competitor entering portfolio company's geography |
| **Product Expansion** | Competitor adding competing features |
| **Customer Wins** | Competitor winning in target verticals |
| **Funding Events** | Competitor raising growth capital |
| **Partnerships** | Competitor forming strategic alliances |
| **Acquisitions** | Competitor acquiring capability/market share |

### 7.4 Relevance Scoring

Items are scored 0-100 on actionability:

| Score Range | Interpretation |
|-------------|----------------|
| **80-100** | Highly actionable, immediate relevance |
| **65-79** | Strong signal, worth attention |
| **55-64** | Moderate relevance, included in newsletter |
| **Below 55** | Filtered out unless carve-out opportunity |
